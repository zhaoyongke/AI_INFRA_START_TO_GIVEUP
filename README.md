> *从"我要改变世界"到"我想改变行业"的完整心路历程*

## 📖 关于本书

《AI Infra 从入门到放弃》是一本面向 AI 基础设施工程师的系统化技术书籍，涵盖从 GPU 硬件基础到大规模分布式系统设计的完整知识体系。

在 AI 大模型时代，模型训练和部署的复杂度呈指数级增长。如何高效地训练一个千亿参数模型？如何让推理服务既快又省？当显存爆炸、NCCL 超时时该怎么办？这些问题构成了 AI Infra 工程师的日常工作。

本书从入门概念讲起，逐步深入推理优化、模型并行、量化、CUDA 编程等核心技术，同时包含大量实战案例、故障排查指南和成本优化策略。无论你是刚入行的工程师，还是想系统提升的从业者，都能从中找到有价值的内容。

**核心观点：**
- AI Infra 是跨学科的领域，需要硬件、算法、系统综合知识
- 没有银弹，优化是权衡的艺术
- 深入原理比记忆配置更重要
- 解决问题的能力 > 掌握知识的多少
